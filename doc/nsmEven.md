# nsmEven()
**not smaller even**

Essa função primeiro verifica entre dois parâmetros qual deles tem o valor menor, depois ele verifica se o menor núemro é par ou ímpar, retornando false para par e true para ímpar.
## exemplo1:

```
print(smEven(32 , 10))
```
**Saída:**
```
False
```
## exemplo2:

```
print(bgEven(25 , 16))
```
**Saída:**
```
False
```

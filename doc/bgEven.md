# bgEven()
**bigger even**

Essa função primeiro verifica entre dois parâmetros qual deles tem o valor maior, depois ele verifica se o maior núemro é par ou ímpar, retornando true para par e False para ímpar.
## exemplo1:

```
print(bgEven(32 , 10))
```
**Saída:**
```
True
```
## exemplo2:

```
print(bgEven(25 , 16))
```
**Saída:**
```
False
```

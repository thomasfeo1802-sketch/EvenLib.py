# nbgEven()
**not bigger even**

Essa função primeiro verifica entre dois parâmetros qual deles tem o valor maior, depois ele verifica se o maior número é par ou ímpar, retornando False para par e True para ímpar.
## exemplo1:

```
print(nbgEven(32 , 10))
```
**Saída:**
```
False
```
## exemplo2:

```
print(nbgEven(25 , 16))
```
**Saída:**
```
True
```

# min_even()

Essa função primeiro verifica entre dois parâmetros qual deles tem o valor menor, depois ele verifica se o menor núemro é par ou ímpar, retornando true para par e False para ímpar.
## exemplo1:

```
print(min_even(32 , 10))
```
**Saída:**
```
True
```
## exemplo2:

```
print(min_even(25 , 16))
```
**Saída:**
```
True
```
